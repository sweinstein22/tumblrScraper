# tumblrScraper

A Tumblr Image Scraper designed to work based off of a query for a certain number of posts or within a certain time frame.  This scraper works to pull posts from a particular tag, and creates two CSVs - one containing just the URLs of the images found for each image post, and one containing the timestamp, date, image urls, captions, and tags for relevant posts.

Run using the command python tumblr_python_scraper.py and follow the prompts
